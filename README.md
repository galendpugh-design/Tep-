# The Everywhere Project Website

A modern, responsive website for The Everywhere Project (TEP), a Philadelphia-based 501(c)(3) harm reduction nonprofit.

## About TEP

The Everywhere Project expands access to harm reduction services across Philadelphia through:

- Community meal sites (Wednesdays at Love Park, Saturdays in Kensington)
- Mobile outreach 6-7 days per week
- Naloxone and fentanyl testing strips
- Safer use supplies and education
- Extreme weather response
- Holiday events and specialized support

Since 2021, TEP has served 1,000+ neighbors every week and distributed 5+ million meals.

## Website Features

- **Multi-page design** with tabbed navigation
- **Responsive layout** for desktop and mobile
- **Service information** including meal sites, mobile outreach, event support
- **Volunteer calendar** with upcoming events
- **Donation integration** with giving levels tied to real outcomes
- **Contact form** for partnerships and inquiries
- **Design system** with custom color palette (purple, teal, cyan)

## Pages

- **Homepage** - Hero, stats, services overview, story timeline
- **Services** - Detailed breakdown of all TEP programs
- **About** - Organization story, mission/vision, team
- **Resources** - Downloadable guides on naloxone, harm reduction education
- **Get Involved** - Volunteer options and full calendar
- **Donate** - Giving levels and impact metrics
- **Contact** - Form and contact information
- **Design Notes** - Rationale for visual design choices

## Design

**Color Palette:**

- Purple (#8b5fbf) - Actions, CTAs, accents
- Teal (#4a7fa0) - Gradients, primary color
- Cyan (#4dd0e1) - Warmth, humanity, highlights
- Ink (#1a1714) - Text
- Sand (#f0f0f8) - Backgrounds

**Typography:**

- Serif (Lora) - Headlines, literary feel
- Sans (Inter) - Body, clean and readable

## Technology

- Pure HTML5, CSS3, and JavaScript (no build tools required)
- Google Fonts for typography
- CSS Grid and Flexbox for layout
- Single-file architecture for easy deployment
